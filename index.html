Remote Controlled Robot with Simultaneous Localization and Mapping (SLAM) (2024)

Description:
Led a team in the design and development of a state-of-the-art remote-controlled search and rescue robot.

Problems Addressed:

This project addresses the need for advanced navigation and mapping capabilities in search and rescue missions, enhancing efficiency and safety.

Technical Skills:
The project was built using C++, Arduino on Raspbian OS, ROS (Robot Operating System), and RPLIDAR. The robot includes the following functionalities:

  • 4 Directional Movement
  • Custom Display Text on LCD
  • Color Identification
  • Mapping using Lidar and RViz
